# Tic Tac Toe Game 🎮

A simple Tic Tac Toe game built using **HTML, CSS, and JavaScript**.
This is a browser-based 2-player game where players take turns marking **X** and **O** on a 3×3 grid.

---

## 📌 Features

* Two-player mode (Player O and Player X)
* Winner detection
* Draw detection
* Reset Game button
* New Game button
* Sound effects for win and draw
* Responsive design
* Hover animations

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Responsive Units)
* JavaScript (DOM Manipulation, Event Handling)
* Downloaded audio files for sound effects

## 🎯 Game Rules

* Player O starts first.
* Players take turns placing their marks.
* First player to get 3 marks in a row (horizontal, vertical, or diagonal) wins.
* If all boxes are filled and no winner → Draw.
* Use **Reset Game** to restart the board.
* Use **New Game** after a winner is declared.
